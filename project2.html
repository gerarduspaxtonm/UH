Tentang Project Ini

Project Website Web Table merupakan website sederhana yang dibuat menggunakan Bootstrap untuk menampilkan data siswa dalam bentuk tabel secara lebih terstruktur dan mudah dibaca. Project ini memanfaatkan berbagai komponen Bootstrap seperti table styling, typography, color utilities, dan layout sederhana untuk menghasilkan tampilan yang lebih rapi dan menarik.

Melalui project ini, saya mempelajari cara menyusun informasi dalam bentuk tabel yang jelas serta memahami pentingnya desain visual dalam penyajian data. Penggunaan Bootstrap juga membantu mempercepat proses pengembangan website sekaligus menjaga tampilan tetap responsif pada berbagai ukuran layar.
