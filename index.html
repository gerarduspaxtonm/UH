Tentang Saya

Saya adalah siswa kelas XI yang memiliki minat besar dalam bidang teknologi, pemrograman, dan pengembangan website. Selain mempelajari HTML, CSS, dan JavaScript dalam pelajaran TIK, saya juga aktif menekuni competitive programming untuk melatih kemampuan logika, pemecahan masalah, dan berpikir algoritmik. Saya memiliki pengalaman mengikuti kompetisi informatika dan terus mengembangkan kemampuan pemrograman melalui latihan serta berbagai tantangan coding.

Melalui pembelajaran TIK dan pemrograman web, saya ingin menggabungkan kemampuan teknis dan kreativitas untuk membuat website yang menarik, interaktif, dan bermanfaat, sekaligus mempersiapkan diri untuk masa depan di bidang teknologi.
